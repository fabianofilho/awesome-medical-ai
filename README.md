## Papers

*   [A deep learning model for clinical outcome prediction using longitudinal inpatient electronic health records](https://academic.oup.com/jamiaopen/article/8/2/ooaf026/8110091) - A Transformer-based, Encounter-level Clinical Outcome model (TECO) for predicting clinical outcomes from EHR data.
*   [Large language models encode clinical knowledge](https://www.nature.com/articles/s41586-023-06291-2) - Evaluation of Med-PaLM, a large language model for medicine, on medical competency exams.
*   [Deep learning-based image classification for integrating pathology and radiology in AI-assisted medical imaging](https://www.nature.com/articles/s41598-025-07883-w) - A framework for integrating multi-modal data in medical imaging.
*   [Generative AI and Foundation Models in Radiology](https://pubs.rsna.org/doi/10.1148/radiol.242961) - A review of foundation models in radiology.

## Hands-On & Code

*   [aws-healthcare-lifescience-ai-ml-sample-notebooks](https://github.com/aws-samples/aws-healthcare-lifescience-ai-ml-sample-notebooks) - Sample notebooks for healthcare and life sciences AI/ML on AWS.
*   [ageron/handson-ml3](https://github.com/ageron/handson-ml3) - A series of Jupyter notebooks that walk you through the fundamentals of Machine Learning and Deep Learning in Python using Scikit-Learn, Keras and TensorFlow.
*   [NIGMS/AI-ML-For-Biomedical-Researchers](https://github.com/NIGMS/AI-ML-For-Biomedical-Researchers) - A practical, data-centric AI/ML module for biomedical researchers.
*   [DeepPSP/torch_ecg](https://github.com/DeepPSP/torch_ecg) - Deep learning ECG models implemented in PyTorch.

## Tools & Libraries

*   [MONAI](https://monai.io/) - AI Toolkit for Healthcare Imaging.
*   [PyHealth](https://pyhealth.readthedocs.io/) - A Python library for healthcare AI.
*   [John Snow Labs Spark NLP for Healthcare](https://www.johnsnowlabs.com/healthcare-nlp/) - State-of-the-art NLP for healthcare.
*   [ClinicalBERT](https://github.com/EmilyAlsentzer/clinicalBERT) - BERT models for clinical notes.
*   [BioBERT](https://github.com/dmis-lab/biobert) - A pre-trained biomedical language representation model for biomedical text mining.

## Pre-trained Models

*   [Med-PaLM](https://sites.research.google/gr/med-palm/) - A large language model designed to provide high quality answers to medical questions.
*   [BioGPT](https://github.com/microsoft/BioGPT) - A large language model for biomedical literature.
*   [GatorTron](https://github.com/uf-hobi-informatics-lab/GatorTron) - A clinical language model from the University of Florida.
