# Datasets — Prontuários Eletrônicos (EHR)

| Dataset | Descrição | Link |
|---------|-----------|------|
| **MIMIC-IV** | Grande banco de dados público com dados de UTI do Beth Israel Deaconess, cobrindo 2008–2019. | [PhysioNet](https://physionet.org/content/mimiciv/) |
| **MIMIC-III** | Versão anterior do MIMIC com 58.976 admissões de 38.597 pacientes. | [PhysioNet](https://physionet.org/content/mimiciii/) |
| **eICU** | Base de dados multicêntrica com dados de alta granularidade de UTIs nos EUA. | [PhysioNet](https://eicu-crd.mit.edu/) |
| **CPRD** | Banco de dados de atenção primária do Reino Unido com mais de 60 milhões de pacientes. | [CPRD](https://cprd.com/data) |
| **EHRSHOT, INSPECT, MedAlign (2026)** | Três datasets de EHR de-identificados livremente disponíveis para pesquisa. Contêm coletivamente 25.991 pacientes únicos, 441.680 visitas, e 295 milhões eventos clínicos. Dados longitudinais complementando MIMIC. | [Stanford HAI](https://hai.stanford.edu/news/advancing-responsible-healthcare-ai-longitudinal-ehr-datasets) |
| **Top 50+ Open-Source Healthcare Datasets (2026)** | Curadoria atualizada de 50+ datasets públicos e acessíveis para construir classifiers de MRI, detectores de arritmia ECG, ou modelos de NLP clínico. | [Medium](https://shubhangilokhande.medium.com/top-50-open-source-healthcare-datasets-for-ai-in-2026-bd802c58b130) |
